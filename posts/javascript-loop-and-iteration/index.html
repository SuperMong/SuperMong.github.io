JavaScript Loop and Iteration

November 10, 2021 · SuperMong

Same as all ( maybe ) programming languages, JavaScript has some statements for loop:

  • while
  • do…while
  • for
  • for…in
  • for…of
  • label
  • break and continue

Most of them are common in other languages, I only introduce for...in, for...of and label here.

for…in#

The for...in is a statement can traverse all enumerable properties of an object.

for (variable in object) {
  statements
}

The following function can return a string that lists the propertie names and their values:

function list(obj) {
  let result = "";
  for (let i in obj) {
    result += i + ": " + obj[i] + " ";
  }
  return result;
}

For an object person with properties name and sex, list(person) will return “name: SuperMong sex: male “.

Notice: If you want to traverse enumerable properties of an object, for...in is a good choice. But for an array (map, set, and so on), for...in will traverse user-defined properties, not array elements.

for…of#

Dirrerent from for...in, for...of statement traverse values of an object.

for (variable of object) {
  statements
}

The following code can make it clearly:

var arr = [5, 2, 6];
let sum = 0;
for (let n of arr) {
  sum += n;          // The value of sum is 5 + 2 + 6 = 13;
}

label#

The label can identify statements. It always be used with break and continue to control loops.

label:
  statements

A simple example:

let x = 0;
someLoop: while (true) {
  x += 1;
  let y = 0;
  while (true) {
    y += 1;
    if (y === 9) {
      break someLoop;
    }
  }
}
