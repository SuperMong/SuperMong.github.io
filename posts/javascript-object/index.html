JavaScript Object

November 16, 2021 Â· SuperMong

Object is always an important concept in object-based language, and JavaScript is a simple object-based language.

An object is a collection of properties. And a property is an association between a key and a value, the value can be a function, in witch case the property is known as a method.

Creat Object#

There are three ways to creat an object:

  1. Object Initializer:

    var obj = {
    property_1:  value_1,
    property_2:  value_2,
    property_3:  value_3,
};

  2. Object Constructor:

    function Obj(property_1, property_2, property_3) {
    this.property_1 = property_1;
    this.property_2 = property_2;
    this.property_3 = property_3;
}

var myObj = new Obj(p1, p2, p3);

  3. Object.Creat Method: Allow you to create a new object with an old object as the prototype.

    var oldObj = {
    p1: v1,
    p2: v2,
    p3: v3
};

var newObj = Object.Creat(oldObj);

Use Object#

It’s easy to use an object. Just like other languages:

objectname.property

or:

objectname[property]

There are two ways to add properties to objects:

  1. Just add property to this one object, does not affect other objects of same type:

    var obj1 = new Obj();    // An object without property 'color'
obj1.color = "red";      // Add property 'color' to obj1

  2. Add property to all objects of this type:

    var obj2 = new Obj();        // An object without property 'color'
Obj.property.color = null;   // Add property 'color' to all objects of type 'Obj'
obj2.color = 'blue';         // Using new property

Getter and Setter#

Getter is a method to get the value of a specific property, and setter is a method to set the value of a specific property. Getter must be parameterless, setter must have one parameter.

This example shows how to define and use getter and setter:

var o = {
    a: 0,
    get b() {
        return this.a + 1;
    },
    set c(x) {
        this.a = x * 2;
    }
};

console.log(o.a);   // 0
console.log(o.b);   // 1
o.c = 100;
console.log(o.a);   // 200

Use Object.defineProperty to define a declared function as getter or setter:

var o = { a: 0 };

Object.defineProperty(o, {
    "b": { get: function() { return this.a + 1; } },
    "c": { set: function(x) { this.a = x * 2; } }
});

These two ways are same, just choose what you like.
